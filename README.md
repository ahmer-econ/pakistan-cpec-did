# Impact of CPEC on Pakistan's FDI: A Difference-in-Differences Approach

## Overview
A quantitative Difference-in-Differences (DiD) analysis evaluating the macroeconomic impact of the 2015 China-Pakistan Economic Corridor (CPEC) announcement on Pakistan’s Foreign Direct Investment (FDI) relative to regional peers.

## Methodology
- **Estimation Technique:** Difference-in-Differences (DiD) estimator to isolate the treatment effect of the 2015 CPEC announcement.
- **Validation:** Event Study framework applied to validate the parallel trends assumption and ensure econometric robustness.
- **Key Findings:** Concluded with a robust null result, indicating no statistically significant divergence in FDI trajectory attributable to the corridor relative to the control group.
